Identifiers

Označavaju varijable, funkcije, svojstva, argumente... Sastoje se od jednog ili više znakova uz dva vrlo jednostavna pravila:

  1. prvi znak mora biti slovo, underscore ili $
  2. svi ostali znakovi mogu biti slova, brojevi, underscoreovi, $, osim razmaka
numSeconds
_value
getRandomValue

Case sensitive

numseconds
_VALUE
getrandomvalue

Statements

U JavaScriptu svaka naredba završava s ; (točka-zarez). Iako ih možemo izostaviti, nije preporučljivo.

var answer = 42
console.log(answer)

Varijable

// razlika između globalnih i lokalnih varijabli
// prilikom deklaracije var ograničava scope varijable na lokalni scope

function sayHi() {
    var message = 'Hello!';
}

function sayHiToAll() {
    message = 'Hello!';
}

sayHiToAll();
console.log(message);
